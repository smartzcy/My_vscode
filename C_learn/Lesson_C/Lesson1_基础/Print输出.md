格式化输出:
    %d 十进制有符号整数
    %u 十进制无符号整数
    %x 十六进制表示的整数
        十六进制：0x开头
    %o 八进制表示的整数
    %f float型浮点数
    %lf double型浮点数
    %e 指数形式的浮点数
    %s 字符串
    %c 单个字符
    %p 指针的值

    %3d : 要求宽度为3位，如果不足3位，前面空格补齐
    %03d : 要求宽度为3位，如果不足3位，前面0补齐
    %-3d : 要求宽度为3位，如果不足3位，后面空格补齐
    %.2f ： 小数后保留2位
    %5.2f : 一共位5位，后面保留两位小数，前面不足的用空格补齐
